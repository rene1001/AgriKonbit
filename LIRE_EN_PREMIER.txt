⚠️ PRÉREQUIS
───────────────────────────────────────────────────────────────────

Avant de démarrer, assurez-vous que :

  ✓ WAMP/MySQL est démarré
  ✓ Node.js est installé
  ✓ Les ports 3000 et 3001 sont disponibles


🐛 PROBLÈMES COURANTS
───────────────────────────────────────────────────────────────────

Erreur: "Cannot connect to database"
  → Vérifiez que WAMP/MySQL est démarré
  → Vérifiez server/.env

Erreur: "Port already in use"
  → Fermez les processus utilisant les ports 3000/3001
  → Ou changez le port dans server/.env

Erreur 500 sur /api/farmer/orders
  → Relancez: node migrations/verify-fixes.js
  → Redémarrez le serveur backend
